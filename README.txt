
Naming:
offset: Arr of offset in ppm
freq: Arr of frequencies in hz
x: a single point of offset or freq
M0: upper case
B0_shift: upper case
WASSR: upper case
nth_slice: the id of slice used for Zspec (starting from 1, so use nth_slice-1 for arr index)
3D index: [nth_slice, x, y] eg. 15*256*256
